# GitHub Dimmed Custom

A modified version of GitHub Dark Dimmed with Java-focused preferences.

## What changed

- Less token highlighting (e.g., method calls, only one color for keywords, no coloring for types, etc)
- Yellow annotations, green javadoc
- No `var` highlight since Java extension infers its actual type instead of simply defaulting to `keyword` (sucks)
